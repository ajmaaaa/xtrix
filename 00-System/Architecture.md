# Architecture

- Interface: Telegram + OpenClaw Web
- AI Brain: Gemini, ChatGPT Free, OpenRouter
- Automation: n8n
- Memory: PostgreSQL
- Second Brain: Obsidian
- Knowledge: Markdown + RAG (bertahap)

Arsitektur bersifat modular sehingga setiap komponen dapat diganti tanpa mengubah keseluruhan sistem.
